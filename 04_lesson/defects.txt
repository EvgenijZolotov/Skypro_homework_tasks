Дефект 1: Метод capitilize()
Тип: Обработка ошибок
Серьезность: High
Описание: Не обрабатывает None значение, вызывает AttributeError
Шаги воспроизведения:
1. Вызвать utils.capitilize(None)
Ожидаемый результат: Должен возвращать None или явное исключение ValueError
Фактический результат: AttributeError: 'NoneType' object has no attribute 'capitalize'

Дефект 2: Метод is_empty()
Тип: Логика работы
Серьезность: Medium
Описание: Возвращает True для строки из пробелов
Шаги воспроизведения:
1. Вызвать utils.is_empty("   ")
Ожидаемый результат: False (строка содержит непечатные символы)
Фактический результат: True
Рекомендация: Использовать string.isspace() для проверки

Дефект 3: Метод list_to_string()
Тип: Обработка None
Серьезность: High
Описание: Не обрабатывает None как входное значение
Шаги воспроизведения:
1. Вызвать utils.list_to_string(None, ",")
Ожидаемый результат: Должен возвращать "" или явное исключение TypeError
Фактический результат: TypeError: 'NoneType' object is not iterable
